# 🏮 潮汕家庭旅遊 2026

**Chaoshan Family Trip | 27–29 March 2026**

A Chinese-language static travel website built for a family trip to Chaoshan (Chaozhou & Shantou), Guangdong, China.

🌐 **Live Site:** [kpchu.github.io/chaosen-trip](https://kpchu.github.io/chaosen-trip/)

## 📅 Itinerary

### Day 1 — Friday 27/3 · 潮州 Chaozhou
- 🚆 08:48 HK West Kowloon → Chaoshan Station 11:11
- 🏨 Check-in: 潮州西湖雅頓假日酒店
- 🍲 Lunch: 潮之鋭記砂鍋粥·潮汕生腌·魚生·獅頭鵝 (牌坊街店)
- 🏛 潮州古城 & 牌坊街
- 🌉 廣濟橋 (Guangji Bridge)
- ✨ Guangji Bridge Light Show (20:00 & 21:30)
- 🥩 Dinner: 官塘兄弟潮汕牛肉店 (牌坊街店)
- 🌙 Night market: 西馬路夜市

### Day 2 — Saturday 28/3 · 汕頭 Shantou
- 🚗 Chaozhou → Shantou (~1 hour)
- 🍵 Lunch: 賞亭樓·潮汕菜·手工茶點·潮汕工夫茶
- 🏙 汕頭小公園 Historic District
- 🏨 汕頭旅社 (Historic Building)
- 🛍 汕頭萬象城
- 🌊 海濱路 Seafront Walk
- 🦆 Dinner: 春梅里鹵鵝館 (龍眼南店)
- 🌙 Night market: 龍眼南路
- 🚗 Return to Chaozhou hotel (~1 hour)

### Day 3 — Sunday 29/3 · 潮州 → 香港
- 🌿 潮州西湖公園 morning walk
- ☕ Breakfast/Tea: 六順茶居
- 🍽 Lunch: 府城四爺
- 🚗 Hotel → Chaoshan Station (~40 min)
- 🚆 15:34 Chaoshan → HK West Kowloon
